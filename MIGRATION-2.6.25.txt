# VibePetCore 2.6.25

## What changed
- Core forge rarity upgrades now use an immediate save/rollback contract.
- If player-data save fails after donor eggs are consumed, donor eggs and the active pet rarity/state are restored instead of being silently lost or desynced.
- Added a regression test for forge save-failure rollback.

## Server action
