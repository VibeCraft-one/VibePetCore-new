# VibePetCore Test Checklist

Короткий чеклист перед реальными тестами на сервере. Идти сверху вниз, без добавления новых фич во время проверки.

## 1. Призыв и предмет

- Выдать новое ядро питомца.
- Положить ядро во вторую руку.
- Нажать ПКМ и дождаться призыва.
- Проверить, что ядро стало активной кнопкой.
- С активной кнопкой во второй руке проверить, что в инвентаре нет обычного пустого ядра как ложного сигнала.
- Во время таймера повторно нажать ПКМ: отсчёт не должен сбрасываться, новые эффекты не должны спамиться.
- Снять предмет из второй руки: питомец должен скрыться.
- Вернуть предмет во вторую руку и снова призвать.
- Перейти в creative и spectator: питомец должен убраться обратно в ядро, активная кнопка не должна оставаться живой.
- В creative попробовать перетащить/клонировать/поменять местами ядро: не должно оставаться дубликатов одного `petId`.

## 2. Режимы питомца

- `/pet call`: питомец возвращается ближе.
- `/pet stay`: питомец держится рядом, но не считается оставленным дома.
- `/pet follow`: питомец снова следует.
- `/pet defense`: боевые реакции включаются/выключаются.
- `/pet autoloot`: работает только у подходящих питомцев.
- `/pet position` и кнопка `Позиция`: циклично меняют место питомца рядом с игроком.
- В GUI питомца кнопка `Режим движения` переключает `рядом` / `следует`.

## 3. Движение

- Пройти по траве, ступенькам, склону, воде рядом, узкому проходу.
- Проверить, что питомец прыгает/обходит мелкие блоки.
- На Э1 проверить обычный блок перед питомцем: он должен прыгать заранее и перепрыгивать, а не упираться в грань.
- Проверить позиции: впереди справа, впереди слева, справа, слева, сзади, ближе, дальше.
- Убежать далеко: питомец должен безопасно вернуться или телепортироваться рядом.
- Сменить мир и вернуться.

## 3.1. Живость

- Постояв AFK 20-40 секунд, проверить короткие действия питомца: смотрит, отдыхает, любопытствует, изучает предмет/траву.
- Держать еду/редкий ресурс в руке: питомец иногда подходит и изучает предмет.
- Смотреть на питомца: action bar показывает короткую подсказку и базовый статус.
- Проверить пассивные эффекты на Э1: должен быть один ключевой эффект коротким окном, потом пауза, без постоянного спама обновления.
- Проверить Э4-Э5: основные эффекты могут держаться почти постоянно, но не больше 3 эффектов одновременно.
- Открыть help по каждому питомцу: вместо абстрактной роли должны быть `Польза` и конкретные бафы по эволюциям.
- Поставить рядом двух игроков с питомцами: иногда появляются мирные взаимодействия без драки питомцев.
- Один раз ударить чужого питомца: прямого урона по entity нет, питомец должен насторожиться.
- Повторно спровоцировать питомца или продолжить PvP между владельцами: должна открыться короткая controlled-схватка питомцев без бесконечной цепочки агра.

## 4. Прогресс

- Покормить питомца подходящей едой.
- Проверить лимит: еда не должна давать больше 20% прогресса уровня за раз.
- Открыть GUI эволюции и проверить, чего не хватает по связи, ресурсам и квестам стадии.
- Положить ресурсы эволюции в инвентарь игрока и в рюкзак питомца.
- Для Э2 проверить порог `ур.3`, `связь 4/10` и `1 из 5` заданий стадии.
- Для Э3 проверить порог `ур.5`, `связь 6/10` и `1 из 5` заданий стадии.
- Для Э4 проверить порог `2 из 5` заданий стадии.
- Для Э5 проверить порог `3 из 5` заданий стадии.

## 5. Рюкзак и лут

- `/pet vault` открывает рюкзак активного питомца.
- Предметы остаются после закрытия/открытия.
- У лутера образец в рюкзаке задаёт фильтр.
- Проверить автоподбор у Allay, Fox, Bee, Bat и Parrot.
- Автоподбор не трогает запрещённые предметы.

## 6. Смерть и рестарт

- Умереть с активным питомцем.
- Проверить штраф восстановления в лоре/баре.
- Перезайти на сервер.
- Перезапустить сервер.
- Проверить, что питомец, ядро и активная кнопка не потерялись.

## 7. Админка

- `/vpc admin inspect` показывает понятное состояние.
- `/vpc admin fixpet` помогает при пропавшем runtime-питомце.
- `/vpc DANGER-DELETE <player>` требует `/vpc confirm` и чистит питомцев игрока.
- `/vpc DANGER-DELETE All` использовать только на тестовом мире или перед осознанной чисткой.
- `/vpc master set` ставит встроенного Пет-Мастера на текущую точку.
- `/vpc master style <sage|keeper|summoner>` меняет внешний стиль и пересоздаёт мастера.
- Рядом с мастером открывается `/pet box open`, и вокруг мастера проигрывается каст-эффект.
- `/vpc lootbox set <arcane|nature|royal>` ставит свой алтарь лутбокса на текущую точку.
- Клик по самому алтарю лутбокса открывает коробку без ручного блока.
- С ядром питомца во второй руке нажать ПКМ по Источнику: открывается GUI Источника, призыв питомца не стартует.
- Быстрый двойной клик по Источнику/алтарю не должен выдавать два яйца за один короткий дубль события.
